# エージェント・チケット（Shared Ticket Board）

ここは、PM（メインエージェント）と各サブエージェントがやり取りするための「チケット/掲示板」領域です。
タスクの依頼、完了報告、ブロック要因、注意事項などの動的なコミュニケーションを書面で非同期に行います。
